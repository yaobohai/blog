## 在本地运行

安装npm

```shell
略
```
安装cnpm

```shell
npm install -g cnpm --registry==https://registry.npm.taobao.org
```

安装hexo

```shell
sudo cnpm install -g hexo-cli
```

运行hexo

```shell
$ hexo s
```
